## simple-shell
simulate ubuntu terminal using C

## How to use
1.Change directory to shell.c.
`cd <directory>`
1.Use GCC to compile shell.c into binary file.
`gcc shell.c -o shell`
1.step-02: Run binary file.
`./shell`
