# simple-shell
simulate ubuntu terminal using C

#How to use
<ol>
<li>Change directory to shell.c<br>
cd <directory></li>
<li>Use GCC to compile shell.c into binary file<br>
`gcc shell.c -o shell`</li>
<li>step-02: Run binary file<br>
`./shell`</li>
</ol>
