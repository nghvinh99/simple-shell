## simple-shell
simulate ubuntu terminal using C

## How to use
1. Change directory to shell.c.    
`cd <directory>`
2. Use GCC to compile shell.c into binary file.    
`gcc shell.c -o shell`
3. step-02: Run binary file.   
`./shell`
